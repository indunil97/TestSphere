# 🌍 TestSphere

**TestSphere** is a marketplace platform that connects clients with certified environmental testing laboratories and service providers. It simplifies the process of requesting, managing, and fulfilling environmental testing services such as air quality, water analysis, emissions monitoring, and noise assessments.

---

## 🚀 Overview

TestSphere enables industries, consultants, and organizations to easily find and collaborate with trusted environmental testing providers. The platform bridges the gap between demand and supply, ensuring efficient service delivery and regulatory compliance.

---

## 🎯 Key Features

### 👤 Client Features

* Create and manage testing requests
* Specify test type (air, water, noise, emissions, etc.)
* Receive and compare quotations from providers
* Track request status
* Review and rate service providers

### 🏢 Provider Features

* Create a professional profile
* Showcase certifications (e.g., ISO 17025)
* View available testing requests
* Submit quotations
* Manage ongoing and completed jobs

### 🔗 Platform Features

* Two-sided marketplace architecture
* Smart request-to-provider matching
* Messaging / communication system
* Review and rating system
* Location-based service discovery *(future scope)*

3. Providers submit quotations

## ⚙️ Getting Started (Development)

### Prerequisites

* Node.js (v18+)
* npm or yarn
* PostgreSQL / Firebase

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/testsphere.git

# Navigate into the project
cd testsphere

# Install dependencies (example)
cd backend
npm install

cd ../frontend
npm install
```

---

## ▶️ Running the Project

```bash
# Start backend
cd backend
npm run dev

# Start frontend
cd frontend
npm run dev
```
